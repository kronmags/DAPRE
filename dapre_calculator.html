Set Weight Reps Notes

How DAPRE Works:

  • Working Weight: Approximately 75% of your 6RM for Set 3 (adjusted to the nearest 5 lbs/kg for simplicity).
  • Set 1: 50% of working weight, 10 reps.
  • Set 2: 75% of working weight, 6 reps.
  • Set 3: Full working weight, max reps (typically 6-10).
  • Set 4: Adjusted weight based on Set 3 reps, max reps.
  • Adjustment Rules for Set 4:
    • 0-2 reps in Set 3: Decrease by 5-10 lbs/kg.
    • 3-4 reps: Decrease by 0-5 lbs/kg.
    • 5-7 reps: Same weight.
    • 8-12 reps: Increase by 5-10 lbs/kg.
    • 13+ reps: Increase by 10-15 lbs/kg.
  • Next Session: Use Set 4 weight as the new working weight if reps in Set 3 were 5-12.
